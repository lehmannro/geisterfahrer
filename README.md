Setup
===============
* sudo apt-get install python-virtualenv
* sudo apt-get install python-pip
* pip install django


Python virtual environment
=================
* virtualenv venv
* source venv/bin/activate


Run
===========
* python manage.py runserver
* python manage.py syncdb
